Version 1 (2006)

Version 1 was my first attempt at creating a personal website in middle school. It was created using Yahoo! PageBuilder and hosted on Yahoo! GeoCities (which was shut down in 2009). Unfortunately, only a picture remains.

See the picture of Version 1

Version 2 (Aug. 2006 - 07)

Version 2 was created in iWeb using the built-in iWeb 1.1 theme "Gazette." It was hosted on GeoCities. Besides the homepage, only two pages were recovered from GeoCities: "About" and "Grades."

Explore Version 2

Version 3 (Aug. 2007 - 09)

Version 3 was my last site hosted on GeoCities. It used the Crimson-PRO iWeb template from 11Mystics.com (which is also no longer hosted). I added a lot of content, such as videos and humor compiled throughout high school.

Explore Version 3

Version 4 (Aug. 2009 - 11)

After GeoCities shut down in 2009, I migrated to Webs (formerly Freewebs) for Version 4 of my website. I added some Song Contest material, and migrated most of Version 3's content to Webs' built-in applications.

Go to Version 4 on Webs

Version 5 (Aug. 2011 - May 2012)

After taking Jerry Cain's SSEA class on JavaScript, for the first time I decided to code my website by hand using HTML5 and CSS3. It was hosted by Stanford. I added my resume and a few fun math proofs.

Explore Version 5

Version 6 (June 2012 - Aug. 2013)

By using the official Stanford Modern template shared by all Stanford websites, Version 6 was the most professional version yet. I added my Stanford four-year-plan, records from Kamehameha High School, and a comprehensive website archive.

Explore Version 6
